## Detailed Usage

EMCCLIFCDriver (EMC VNX FC driver) is based on the FibreChannelDriver  
defined in Block Storage.

EMCCLIISCSIDriver (EMC VNX iSCSI driver) is based on the ISCSIDriver defined in 
Block Storage.
